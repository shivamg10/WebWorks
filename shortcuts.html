Some of the Commonly used Shortcuts

  • Reloding the html page : ctrl+f5 or ctrl+r

  • Copying file path : shft+alt+c

  • Saving the file : Ctrl+S

  • Switching windows : Alt + Tab

  • Auto Indent HTML : Shft+Alt+F

  • Swicthing to next tab in chrome : Ctrl + Tab

  • Switching to Prev tab in chrome : Ctrl+ Shft+ Tab
